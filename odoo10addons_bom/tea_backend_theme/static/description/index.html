Tea Backend Theme

The first Odoo 10.0 community back-end sidebar and application drawer theme
  • It's very efficient when it runs in different terminals
  • Under the PC terminal, the sidebar is added to allow for quick work.
  • Under the mobile terminal, use the application drawer.
  • Beautiful login page.

How to Custom

  • Custom Login Page Background Picture

    Found in the /static/src/js/login.js file 
    $.backstretch([
    "/tea_backend_theme/static/src/img/bg/1.jpg",
    "/tea_backend_theme/static/src/img/bg/2.jpg",
    "/tea_backend_theme/static/src/img/bg/3.jpg",
    "/tea_backend_theme/static/src/img/bg/4.jpg"
    ], {
        fade: 1000,
        duration: 8000
        }
    );

  • Custom Application Drawer Background Picture

    Found in the /static/src/less/app_drawer.less file 
    // -----------------------------------
// App Drawer Panel Background Image
// -----------------------------------
.drawer-nav{
	background: transparent url('/tea_backend_theme/static/src/img/drawer-background2.png') no-repeat scroll center center / cover;
}
